TrollBot's Command List

The available command for TrollBot are:

Help - It shows a list of commands and the link to this website.

Kick - It kicks a user (Syntax: "Kick @user")

Ban - It bans a user(Syntax: "Ban @user")

Hey - It greets the user back.

Troll - Trolls you :)

Add - Multiplies numbers(Syntax: "Add number number")

Multiply - It adds number(Syntax: "Multiply number number")

Say - The bot says what you say, there's a little surprise there :)(Syntax: "Say message")

Mute - It mutes the user(Syntax: "Mute @user")

Adm - Only for admins,it can give the admin role.(Syntax: "Adm @user")

Bot - Only for Admins, it gives the bot role.(Syntax: "Bot @user")

The prefix of the bot is "<". It must be put before each command.
